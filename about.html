About ConnectHub

A purpose-driven platform designed for meaningful and respectful connections.

Explore Purposes

Problem

Many social platforms promote random or unclear matching. This often leads to low-quality interactions and confusion about user intent.

Solution

ConnectHub organizes connections by clear purpose such as academics, emotional support, professional growth, and long-term intent.

Objectives

  • Provide clarity through intent-based connection categories
  • Promote respectful and safe interaction
  • Design a modern UI using HTML and CSS
  • Present information clearly using forms and tables

Technology Used

  • HTML5 (semantic structure)
  • CSS3 (grid, layout, glassmorphism)
  • Responsive design for mobile and desktop
